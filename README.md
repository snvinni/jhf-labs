# JHF Labs

Site do estudio JHF Labs — Just Have Fun.

## Stack

HTML + CSS + JS vanilla. Zero build.

## Deploy

Hospedado no GitHub Pages. Push na branch `main` → Settings → Pages → Source: `main` / `/ (root)`.

## Estrutura

```
index.html     # Pagina unica
style.css      # Estilos
assets/        # Screenshots dos apps
```

## Editar

- **Take One**: trocar descricao em `index.html` (section `#projetos`, artigo 02)
- **Screenshots Take One**: substituir `assets/takeone-1.svg`, `-2.svg`, `-3.svg` por PNGs reais (e atualizar extensao no HTML)
- **Links de download**: preencher `href="#"` nos botoes
